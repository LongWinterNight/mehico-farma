Промпт для заголовков тизеров

NATIVE ADS

Генерация цепляющих заголовков для тизерной рекламы нутра-офферов 

Действуй как эксперт по нативной рекламе для LATAM рынка.

Создай 15 цепляющих заголовков для тизерной рекламы 
нутра-оффера (суставы/похудение/паразиты).

Требования:
1. Используй триггеры любопытства
2. Избегай прямых обещаний лечения
3. Адаптируй под местный менталитет
4. Длина: 45-65 символов
5. Добавь эмоциональный крючок

Формат: таблица с колонками [Заголовок, Триггер, Длина]

Промпт для Facebook Ads

FACEBOOK ADS

Создание рекламных текстов для Facebook/Instagram с обходом модерации 

Ты — специалист по рекламным креативам для Facebook Ads.

Напиши 7 вариантов описания для рекламы нутра-продукта.

Структура:
1. Боль/проблема ЦА (2-3 предложения)
2. Эмпатия
3. Решение (без прямых обещаний)
4. Social proof
5. CTA

Требования:
- Тон: дружеский, доверительный
- Избегай запрещённых слов
- 2-3 эмодзи
- Длина: 125-175 символов

Промпт для UGC-сценария

VIDEO CONTENT

Создание сценариев для коротких вертикальных видео TikTok/Reels 

Создай сценарий для 30-секундного UGC-видео TikTok/Reels.

Таймлайн:
├─ 0-3 сек: HOOK
├─ 3-15 сек: PROBLEM
├─ 15-25 сек: SOLUTION
└─ 25-30 сек: CTA

Для каждого блока:
- Ракурс камеры
- Текст на экране
- Действия актёра
- Аудио

Формат: таблица с таймкодами
